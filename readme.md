This is a small project that creates:
1. An API that has routes to :
    * register a user
    * Register a business
    * Upload and save a CSV file
    * Provide to the logged in user information about their Top items produced in terms of
    value for a given time period, default being last 30 days
    * Provide to the logged in user information about their Top items produced, in terms of
    quantity produced,for a given time period, default being last 30 days
    * Provide to the logged in user information about amount incoming and amount out going
    for a given time period, default being last 30 days

2. A detached frontend which shows
    * A simple bar chart that show the top five Items in terms of quantity
    * A simple bar chart that show the top five Items in terms of value
    * Pie chart that shows relationship between the amount of money coming in and amount going out
